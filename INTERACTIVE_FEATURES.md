# Interactive Features Added 🎨

Your Softech website is now fully interactive with modern animations and effects!

## 🎯 Interactive Portfolio

### Click to View Details
- Click any portfolio card to see full project details
- Modal popup with:
  - Full project description
  - Key metrics and results
  - Technology tags
  - High-quality images

### Hover Effects
- Image zoom on hover
- Gradient overlay appears
- "View Details →" text slides in
- Smooth card lift animation

## 📊 Animated Statistics Counter

### About Section
- Counters animate when scrolled into view
- Shows:
  - 150+ Projects Completed
  - 50+ Happy Clients
  - 98% Average Accuracy
- Cards lift on hover
